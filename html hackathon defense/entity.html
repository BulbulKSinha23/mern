Entity in HTML:

A "symbol entity" refers to a special sequence of characters used to represent a character that cannot be easily typed directly or might be interpreted as part of the HTML markup itself. These are also known as HTML character entities.

There are two primary ways to represent symbol entities:

Named Entities: These use a name for the character, preceded by an ampersand (&) and followed by a semicolon (;).

Numbered Entities: These use the decimal or hexadecimal Unicode value of the character, preceded by an ampersand (&) and a hash symbol (#), and followed by a semicolon (;).
