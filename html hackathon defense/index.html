  • DAY- :- Introduction to Web Development.

    • How does Network Work?

    a web browser sends an HTTP request to a web server, which responds by sending the website's data (HTML, CSS, etc.) back in packets across the network (primarily TCP/IP) to be reassembled and displayed by the browser. This process involves a series of steps including DNS lookups to find the server's IP address, routing packets to the correct destination, and reassembling the data at the client end to form the web page.

    Here's a breakdown of the process:

    Types of Address:

    1. IP Address
    2. MAC Address

    Port Number

    A port number is a numerical identifier (like 0-65535) on a network device that directs incoming data to a specific application or service, similar to how an apartment number directs mail to a specific resident in a building. While an IP address identifies a device, a port number identifies a particular software application or service running on that device, ensuring the data goes to the correct program, such as web browsing (port 80) or secure web browsing (port 443).

  • DAY:- What is HTML?

    • WEB DEVELOPMENT

    HTML, CSS, and JavaScript are the three core technologies used to build web pages and web applications. They work together to define the structure, presentation, and behavior of content on the web.

    HTML (HyperText Markup Language):

    HTML is the standard markup language for creating web pages. It provides the structure and content of a webpage using a system of elements and tags. These tags define various parts of the content, such as headings, paragraphs, images, links, and more, creating the basic framework upon which other technologies build.

    CSS (Cascading Style Sheets):

    CSS is a style sheet language used for describing the presentation and visual formatting of a document written in HTML. It controls aspects like colors, fonts, layout, spacing, and how elements are displayed on different screen sizes. CSS separates the content (HTML) from its visual presentation, making it easier to manage and update the design of a website.

    JavaScript (JS):

    JavaScript is a programming language that adds interactivity and dynamic behavior to web pages. It enables features like animations, form validation, real-time updates, and handling user interactions (e.g., button clicks, mouse movements). JavaScript can manipulate the HTML structure and CSS styles of a page, allowing for dynamic content changes and a more engaging user experience.

    Mern Stack

    HTML tags are keywords enclosed in angle brackets (< >) that define the structure and content of a web page. They instruct web browsers on how to render various elements, such as text, images, links, and other media.

    2. Paragraph Tag

  • DAY-:- HTML BoilerPlate, File Path,Div,Class,ID.

    • Boiler Plate code of HTML

    In HTML, "boilerplate code" refers to the standard, foundational set of tags and declarations that are typically included at the beginning of every HTML document. This code provides the basic structure and essential information required for a web page to be correctly interpreted and displayed by a web browser.

    File path System:

    In HTML, you specify a file's location using a file path, which can be either absolute or relative. Absolute paths are complete, while relative paths describe a location in relation to the current file. File paths are used to link to external resources such as images, stylesheets, and other HTML pages.

    Selectors in HTML:

    Selectors in HTML, primarily used within the context of CSS, serve to "find" or select specific HTML elements to apply styles to them. These selectors allow for precise targeting of elements based on various criteria.

    Types of Selectors:

    • Simple Selectors:

      • Target elements based on basic characteristics.

    • Element Selector:

      • Selects all instances of a specific HTML element (e.g., p for all paragraph elements).

    • ID Selector:

      • Selects a unique element based on its id attribute (e.g., #myID). An id should be unique within an HTML document.

    • Class Selector:

      • Selects elements based on their class attribute (e.g., .myClass). Multiple elements can share the same class.

    • Universal Selector:

      • Selects all elements on the page (e.g., *)

  • DAY-:- Forms in HTML:

    • Forms in HTML:

    An HTML form is a section within an HTML document used to collect user input. It is defined using the <form> tag and can contain various interactive controls, such as text fields, checkboxes, radio buttons, dropdown menus, and submit buttons.

    A form allows an organisation to collect a uniform set of data from many parties in a consistent manner. Forms, when completed, vary in their purpose; for example, a form might be a statement, a request, or an order. A cheque may also be considered a form. In addition, there are several forms for taxes.
