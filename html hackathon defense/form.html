Form in HTML:

A webform, web form or HTML form on a web page allows a user to enter data that is sent to a server for processing. Forms can resemble paper or database forms because web users fill out the forms using checkboxes, radio buttons, or text fields.

The basic structure of an HTML form includes:

<form>tag:

This element encloses all the form controls and specifies attributes like action (the URL where the form data is sent) and method (how the data is sent, typically "get" or "post").

Input elements:

These are the interactive controls that allow users to enter data.Examples include <input type="text">, <input type="password">, <input type="checkbox">, <input type="radio">, <textarea>, and <select>.


<label> tag:

Used to associate a descriptive label with an input element, improving accessibility and user experience.

Important <input> types

The type attribute can be changed to collect different kinds of data:

  • type="text": A single-line text field.

  • type="password": A text field that masks the user's input for sensitive information.

  • type="email": A text field that requires the input to be a valid email address.

  • type="radio": Radio buttons for selecting one option from a set.

  • type="checkbox": Checkboxes for selecting zero or more options.

  • type="submit": A button that sends the form data.

  • type="date": creates a field that allows users to select a date, typically through a calendar interface provided by the browser.

  • type="textarea": a textarea allows users to enter large amounts of free-form text.
