Lists in HTML

HTML offers three primary types of lists for structuring content:

Unordered Lists (<ul>):

These lists are used to group a set of related items where the order does not matter. Each list item is typically marked with a bullet point by default.

Ordered Lists (<ol>):

These lists are used to group a set of related items where the order is significant. Each list item is typically marked with numbers, letters, or Roman numerals by default, indicating a sequence.

The Target Attribute:

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

The target attribute specifies where to open the linked document.

The target attribute can have one of the following values:
  • _self- Default. Opens the document in the same window/tab as it was clicked
  • _blank- Opens the document in a new window or tab

Nested List:

Nested lists in HTML allow for the creation of hierarchical structures by placing one list inside another. This is achieved by embedding a <ul> (unordered list) or <ol> (ordered list) element within an <li> (list item) element of a parent list.
